# Test-IsAdmin

## SYNOPSIS
Returns True if session running in elevated context

## SYNTAX

```
Test-IsAdmin
```

## DESCRIPTION
Returns True if session running in elevated context, such as Administrator (Windows) or root (Linux)

## EXAMPLES

### EXAMPLE 1
