# Get-Abbreviation

## SYNOPSIS
Get abbreviation from string of words

## SYNTAX

```
Get-Abbreviation [-String] <String> [<CommonParameters>]
```

## DESCRIPTION
Derive an abbreviation from a string of words

## EXAMPLES

### EXAMPLE 1
```
Get-Abbreviation "Eastern Standard Time"
```

Returns "EST"
